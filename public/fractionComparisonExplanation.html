Decide which fraction is bigger using simple rules.
If two fractions have the same denominator (bottom number), the bigger numerator (top number) means the bigger fraction. Example: \(\tfrac{5}{8} > \tfrac{3}{8}\).
\(\tfrac{5}{8}\)
>
\(\tfrac{3}{8}\)
If two fractions have the same numerator (top number), the smaller denominator means bigger pieces, so the fraction is bigger. Example: 3/5 > 3/7.
\(\tfrac{3}{5}\)
>
\(\tfrac{3}{7}\)
If denominators are different and numerators are different, make a common denominator or compare using equivalent fractions.
\(\tfrac{2}{3} = \tfrac{4}{6}\)
>
\(\tfrac{1}{2} = \tfrac{3}{6}\)
Rules recap:
  • Same denominator ‚Üí bigger numerator wins.
  • Same numerator ‚Üí smaller denominator wins.
  • Otherwise ‚Üí use equivalent fractions with a common denominator.
